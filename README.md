College Event Management Website

A visually appealing and responsive static frontend website designed to showcase and promote college events. This single-page application provides a central hub for students to find information about past, present, and future campus activities, including cultural fests, tech fests, and sports events.

🛠️ Technologies Used
* **HTML5:** For the structure and content of the website.
* **CSS3:** For all styling, layout, animations, and responsiveness.
    * **Flexbox:** Used extensively for creating flexible and responsive layouts.
    * **Keyframe Animations:** Custom animations for a dynamic feel.
* **JavaScript (Vanilla):** Minimal JavaScript used for the `onclick` functionality of the image gallery navigation arrows.
